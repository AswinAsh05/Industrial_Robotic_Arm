# 🤖 Industrial Robotic Arm – Fusion 360 Design

A multi-joint industrial robotic arm designed and simulated using **Autodesk Fusion 360** to study robotic manipulator structure, joint constraints, and motion behavior.

This project focuses on understanding **robotic arm kinematics, mechanical assembly, and motion simulation** using CAD tools.

## ⚙️ Features

• Multi-joint industrial robotic arm design  
• Revolute joints with defined motion limits  
• Mechanical gripper end-effector  
• Joint drive simulation for motion testing  
• Modular components designed for potential **3D printing**

---

## 🛠 Tools Used

- Autodesk Fusion 360
- Mechanical Assembly Constraints
- Joint & Motion Simulation
- Parametric 3D CAD Modeling

---

## 🧠 Learning Objectives

This project was created to understand:

- Robotic arm structure and manipulators
- Revolute joint axes and constraints
- Mechanical linkage motion
- CAD assembly design
- Motion simulation in robotic systems

## 🖨 3D Printing

All parts are exported as **STL files** and can be used for 3D printing.

Recommended settings:

- Units: **Millimeters**
- Material: **PLA / PETG**
- Layer height: **0.2 mm**
- Infill: **20–40%**
